# interactive-product-selector
A responsive product selection interface featuring three interactive boxes that expand to reveal size and color options. Implements a BOGO (Buy One Get One) deal system with dynamic pricing, selection states, and smooth animations. Built without frameworks using clean, semantic HTML, custom CSS, and vanilla JavaScript.

# Interactive Product Selector

A responsive product selection interface with three interactive boxes that expand to reveal size and color options.

## Features
- Three expandable product boxes (top/middle/bottom layout)
- Dynamic size and color selection
- Real-time price calculation
- Responsive design
- Smooth CSS animations
- Pure vanilla JavaScript (no frameworks)

## Live Demo
[View Live Demo](https://shashi6138.github.io/interactive-product-selector/)

## Technologies Used
- HTML5
- CSS3 (Flexbox, CSS Grid, transitions)
- Vanilla JavaScript

## How to Use
1. Click on any product box to expand it
2. Select your preferred size and color
3. The total price updates automatically
4. Click "Add to Cart" to confirm selection
